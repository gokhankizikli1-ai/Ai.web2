# KorvixAI AI Development Guide

## Product Vision
Korvix is an AI Operating System.
The Generation Engine is the core product.

## Development Mode
- Optimize for shipping.
- Think before coding.
- Write production-quality code.
- Validate reasonably.
- Spend no more than 20% of a session on testing.
- Avoid exhaustive regression testing.
- Validate only affected code.

## Scope Discipline
- One sprint = one objective.
- Never expand scope.
- Keep PRs focused.
- Document follow-up ideas instead of implementing them.

## Decision Making
- Choose the simplest solution.
- Avoid clever abstractions.
- Favor readability.

## Product First
Optimize for user experience first.

## UI Philosophy
Target the quality of ChatGPT, Claude, Linear, Notion, Raycast, Apple, Vercel and Lovable.

## Pull Request Standard
- Objective
- Summary
- Files Changed
- User Impact
- Risks
- Validation
- Follow-up Recommendations
